## Pagination Component
페이지네이션을 vue 컴포넌트로 생성

---

#### language & framework
- html
- css
- javascript
- vue.js
- bootstrap

---

#### 예제 실행 방법
1. example 디렉토리 이동
  ```console
  $ cd example
  ```

2. npm install
  ```console
  $ npm install
  ```

3. 로컬 개발용 서버 띄우기
  ```console
  $ npm run dev
  ```
